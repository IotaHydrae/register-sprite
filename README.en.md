# 寄存器小精灵

#### Description
我做了一个查看寄存器位小工具，它是基于python tkinter编写的，具有良好的界面和一些简单的功能，它可以轻松实现10位、16位、8位和2位之间的转换，方便用于学习各种嵌入式寄存器的位配置，嵌入式工程师的必备神器。
我提供了使用pyinstaller构建好的可执行文件，当然你也可以修改后构建自己的EXE文件
